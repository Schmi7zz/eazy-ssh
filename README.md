## вңЁ Features

### рҹ–Ҙ Web Terminal (Mini App)
- **Real terminal** вҖ” Full xterm.js with color, scrollback, and auto-resize

- **Multiple themes** вҖ” Dark, Light, Dracula, Monokai
- **Persistent servers** вҖ” Saved server list survives app restarts
- **Telegram-only** вҖ” HMAC-SHA256 initData validation
- **Native feel** вҖ” Adapts to Telegram's dark/light theme automatically

### рҹ“Ӯ SFTP File Manager
- **Browse files** вҖ” Navigate your server filesystem visually
- **Upload / Download** вҖ” Transfer files directly from Telegram
- **Create folders** вҖ” Make new directories on the fly
- **Rename / Delete** вҖ” Full file management
- **рҹ“қ Built-in Code Editor** вҖ” Edit files directly on the server with Ace editor
  - Syntax highlighting for 40+ languages (Python, Go, JS, Bash, YAML, Docker, etc.)
  - 10 editor themes (Monokai, Dracula, Cobalt, etc.)
  - Undo/Redo, Search (Ctrl+F), Word Wrap
  - Save with Ctrl+S вҖ” writes back to server instantly
  - Line/column indicator, error & warning counter
  - Max 2MB file size for editing

### рҹ“ҹ Chat Terminal (Inline SSH)
- **SSH in chat** вҖ” Run commands directly in Telegram chat, no Mini App needed
- **Live output** вҖ” Terminal output displayed as auto-updating message
- **Control buttons:**

## рҹ“Ӣ Prerequisites

- A **VPS** with Ubuntu 22+ and a public IP
- A **domain name** (Telegram requires HTTPS)
- A **Telegram bot token** from [@BotFather](https://t.me/BotFather)

Docker will be installed automatically if not present.

## рҹҡҖ Quick Install

### Step 1 вҖ” Create DNS Records

Create two A records pointing to your VPS IP:

```
ssh-terminal.yourdomain.com  вҶ’  YOUR_VPS_IP
ssh-api.yourdomain.com       вҶ’  YOUR_VPS_IP
```

> If using Cloudflare, disable the proxy (grey cloud) initially.

### Step 2 вҖ” One-Line Install

```bash
bash <(curl -s https://raw.githubusercontent.com/Schmi7zz/eazy-ssh/main/install.sh)
```

The installer will ask for your domains, bot token, and admin ID, then automatically set up everything.

### Step 3 вҖ” Configure BotFather

After install, go to [@BotFather](https://t.me/BotFather):

1. `/setmenubutton` вҶ’ select your bot вҶ’ URL: `https://ssh-terminal.yourdomain.com` вҶ’ Title: `Open Terminal`
2. `/newapp` вҶ’ select your bot вҶ’ Web App URL: `https://ssh-terminal.yourdomain.com` вҶ’ Short name: `terminal`

### Step 4 вҖ” Done! рҹҺү

Open `t.me/YOUR_BOT/terminal` in Telegram вҶ’ add a server вҶ’ connect!

## рҹ”’ Security

- **Telegram validation** вҖ” Every WebSocket connection validates `initData` via HMAC-SHA256
- **No server-side credential storage** вҖ” SSH credentials are sent per-connection, never persisted
- **Client-side only** вҖ” Server list saved in localStorage within Telegram's webview
- **HTTPS everywhere** вҖ” All traffic encrypted via TLS
- **Origin restriction** вҖ” Optional `ALLOWED_ORIGIN` env var
